# AI Discovery Agent

A lightweight Streamlit app that turns messy B2B discovery notes into a structured deal brief.

## What it does
- Extracts problems, success metrics, requirements, risks, and open questions
- Suggests a tailored demo agenda
- Flags missing information (gap detection)
- Adds deal fit score and confidence level
- Includes a human-review notice (advisory output)
